## Scripts overview

Organized by purpose:

- `scripts/clawd/` — Clawd install/config/ops on sparky hosts.
- `scripts/deployment/` — deploy, sync, and restart backend.
- `scripts/git/` — git + SSH setup for sparkies.
- `scripts/moltbook/` — Moltbook register, post, learn, cron.
- `scripts/testing/` — test runs, workflow tests, suite runner.
- `scripts/utils/` — small utilities (health check, viewer pull, eval).

Run from repo root unless noted in script header comments.
